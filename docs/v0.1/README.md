# v0.1 MVP 阶段文档

v0.1 自用 MVP 按 **6 个阶段**（0–5）交付。每个阶段一份独立文档，记录目标、任务、验收与完成时的实际产出。

| 阶段 | 文档 | 名称 | 状态 |
|------|------|------|------|
| 0 | [phase-0.md](./phase-0.md) | 仓库与契约 | ✅ 已完成 |
| 1 | [phase-1.md](./phase-1.md) | CLI Runtime 内核 | ⬜ 未开始 |
| 2 | [phase-2.md](./phase-2.md) | 可组合 Agent | ⬜ 未开始 |
| 3 | [phase-3.md](./phase-3.md) | Backend 控制面 | ⬜ 未开始 |
| 4 | [phase-4.md](./phase-4.md) | Web 聊天 MVP | ⬜ 未开始 |
| 5 | [phase-5.md](./phase-5.md) | 打磨与自用 | ⬜ 未开始 |

**v0.1 总目标**：一个人、一台（或多台）电脑、一个 Web 页，能组装 agent、直连 CLI 聊天、看 Session 树，API Key 在后端。

总览与后续版本见 [路线规划](../roadmap.md)。快速查看当前进度见 [current-phase.md](../current-phase.md)（指向本目录）。
